Detailed Description

Batch normalization: normalize layers to zero mean/unit variance. See paper: Sergey Ioffe and Christian Szegedy. "Batch Normalization: Accelerating Deep Network Training by Reducing Internal Covariate Shift." ICML 2015. This keeps a running mean and variance (with exponential decay) instead of computing it over the data at test time. This approach seems to have become standard. See also: https://cthorey.github.io/backpropagation/

Set up batch normalization.

Parameters
decayControls the momentum of the running mean/variance averages.
gammaThe initial value for gamma. The paper recommends 1.0 as a starting point, but other papers have had better results with a smaller value (e.g. 0.1).
betaThe initial value for beta. This should almost always stay at zero.
