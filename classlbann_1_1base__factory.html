Detailed Description

template<typename Method>
class lbann::base_factory< Method >

lbann_base_factory.h - base factory from which different methods (for layers, optimizer) could be registered. This class is to be inherrited by appropriate classes e.g., layer_factory, optimizer_factory. Derived classes holds a list of available methods of their types e.g., full connected layer, convultional layer for Layer class
